Manual Test Instructions

  1. Open the main editor
  2. Test canvas navigation:
    • Mouse wheel to zoom in/out
    • Middle-click and drag to pan
    • Use zoom controls in toolbar
  3. Test node creation:
    • Select a color from the creation panel
    • Click "Create Node" or double-click on canvas
    • Verify node appears and can be selected
  4. Test node interaction:
    • Click on a node to select it
    • Verify properties panel updates
    • Drag node to move it
    • Edit properties and verify updates
  5. Test grid functionality:
    • Toggle grid visibility
    • Toggle snap to grid
    • Verify nodes snap when moving
